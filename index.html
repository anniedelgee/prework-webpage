Prework Study Guide

HTML

  • HTML is the structural code for the webpage, into which all other coding that will make up the webpage, such as css and js is linked
  • Metadata is stored in the head element, while the body element presents the content of the page for the user.

CSS

  • Cascading stylesheets contain the aesthetic information of a webpage
  • Graphic design features are all incoroporated into a web applciation here; colour, typography and layout.
  • It would be useful to approach this step with wrieframes or a full UI prototype to hand

GIT

  • GIT is used for collaboration and version control.
  • Using an IDE, Integrated Development Environment, is connected to the remote repository in Github
  • This allows multiple edits to be made to a single web application, by multiple develoeprs in various locations

Javascript

  • Javascript adds functionality to an otherwise purely aesthetic web appliction
  • JS gives webpages usability and interactivity.
